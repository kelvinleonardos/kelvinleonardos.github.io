Cara Clone Repository ke File Explorer

Langkah ini digunakan untuk menyalin repositori ke komputer kita.
Sehingga kita dapat memindahkan file ke dalam repositori melalui salinannya di komputer kita.

  1. Masuk kembali ke Repository yang mau di clone lalu klik code

  2. Klik tanda salin atau salin ling yang tersedia secara manual

  3. Kembali ke file explorer, masuk ke tempat dimana kita akan menyimpan salinan repositori

  4. Pada bagian path atau alamat file, klik, lalu ketik cmd, enter

  5. Nantinya akan muncul cmd dan ketikkan seperti pada gambar(git clone "link repo")
    (Ingat untuk menggunakan tanda kutip pada link)

  6. Anda bisa melihat folder bernama sesuai dengan repositori anda
    dan berisi file dengan nama .git (file .git mungkin akan tidak tampak di beberapa komputer
    namun tidak apa, yang penting foldernya sudah ada
