Cara Buat Repository Baru


  1. Buka GitHub di browser

  2. Klik icon profile di ujung kanan atas

  3. Klik my repository

  4. Klik new

  5. Berikan nama dan deskripsi untuk repository yang akan dibuat
    Bila repository ingin dapat diakses orang lain, maka pastikan pilihan berada di public
    (Secara default, tidak ada yang perlu diubah ubah di bagian ini,
    cukup masukan nama dan deskripsi)

  6. Klik create repository, dan anda sudah berhasil membuat repository baru
    Anda bisa melihat repository baru anda di my repository
