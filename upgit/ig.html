Cara Install Git

Bila sebelumnya anda sudah perhan menginstall Git
(Bila saat asistensi anda sudah pernah juga menggunakannya)
maka langkah ini dapat anda lewati


  1. Masuk ke web yang tertera di gambar, dan klik download
    sesuai dengan sistem operasi yang dimiliki

  2. Tunggu hingga download selesai dan cari file hasil download
    di file explorer

  3. Buka file hasil download dan lakukan instalasi
    Untuk saat ini, anda tidak perlu merubah pengaturan apapun
    cukup tekan next-next sampai proses instalasi selesai

  4. Bila proses instalasi selesai, anda dapat mengecek keberadaan git di komputer anda
    dengan masuk ke cmd dengan cara menekan win + r,
    ketikan cmd pada tempat isian yang tersedia, lalu enter atau klik ok

  5. Ketik "git --version" (tanpa tanda kutip) lalu tekan enter
    Bila proses instalasi berhasil, anda akan mendapat pesan seperti bada gambar
